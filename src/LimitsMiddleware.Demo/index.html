Limits Middleware Demo

Tweak some values and click 'download' (once or multiple times concurrently) to experience a file with limits applied to it.

Max global bandwidth is set at 10MBps. Download several larges files concurrently to see the limit applied; the more concurrent requests, the more each request is limited.

Max number of concurrent requests is 4.

This is best experienced with Firefox with FasterFox Lite and DownThemAll addons. Some browsers my limit number of concurrent requests.

File size:

1 KB 10 KB 100 KB 1 MB 10 MB 100 MB

Max Bandwidth per Request:
